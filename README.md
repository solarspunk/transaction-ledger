# transaction ledger

a simple CLI tool to record and reference deposits and payments

## features

* add deposits and payments to the record
* all information stored in a single csv file
* view all entries from the ledger screen
* ability to filter entries by either deposits or payments, vendor name, previous month, etc.
* each entry contains: 
```date | time | description | vendor | amount```

## usage requirements
* must have a keyboard and CLI
